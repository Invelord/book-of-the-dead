# Book of the Dead Updated to 6000.3.6f1 LTS

Book of the Dead is an older high-fidelity HDRP environment project from Unity Technologies, originally created as a tutorial and based on the Book of the Dead real-time cinematic. It demonstrates advanced lighting, dense foliage, and photorealistic materials. This version has been updated and optimised to run natively on modern M-series Macs using recent versions of macOS and Unity 6.
